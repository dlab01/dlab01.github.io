## 책 메모
- Apps Script 웹앱 + 정적 사이트(github.io) `fetch()` 조합의 함정: 액세스가 "Google 계정 있는 모든 사용자"면 로그인 리다이렉트 때문에 cross-origin fetch가 통째로 깨진다. 반드시 **"모든 사용자(익명)"** 여야 함. daddyslab.com 같은 워크스페이스는 이 옵션을 막을 수 있어 개인 Gmail 배포가 우회책.
- 같은 시트라도 `.xlsx`(업로드) vs 네이티브(`application/vnd.google-apps.spreadsheet`)는 완전히 다름. URL의 `rtpof=true&sd=true`가 .xlsx 신호. 메타데이터 mimeType으로 사전 검증 가능.
- 코드 수정 후 "배포관리→편집"에서 **버전:새 버전**을 안 고르면 옛 코드가 계속 서빙됨 (이번에 한 번 헛걸음).
